# Dental Website (WordPress + Elementor)

## Overview
A responsive dental practice website built using WordPress and Elementor. Designed with a clean, modern layout and user-friendly navigation.

## Features
- Custom homepage layout
- Service sections with icons
- Contact form integration
- Mobile-responsive design
- Elementor-based visual editing

## Tools Used
- WordPress
- Elementor
- Canva (for graphics)
- MAMP (local development)

## My Role
Designed and built the full layout, structure, and visual components. Focused on clean UI, readability, and responsive behavior.
