Dental Website — WordPress + Elementor
Overview
A fully responsive dental practice website built using WordPress and Elementor, designed with a clean, modern layout and a patient‑friendly user experience. The site emphasizes clarity, trust, and easy access to essential services.

Features
Custom hero section with clear call‑to‑action

Service sections with icons and structured layout

About section with professional, approachable tone

Contact and appointment callouts

Mobile‑optimized layout with strong visual hierarchy

Consistent color palette and typography for a modern, boutique feel

Tech Used
WordPress

Elementor Page Builder

Custom CSS for spacing, alignment, and responsive refinements

Canva for visual assets

MAMP for local development

My Role
I handled the full design and build process:

Customized spacing, alignment, and mobile responsiveness

Selected imagery and icons to support a clean, professional aesthetic

Ensured accessibility and readability across devices

What I Learned
How to structure a service‑based website for clarity and conversion

Improving Elementor layouts with custom CSS

Creating a consistent visual identity across multiple sections
